# 臼井靈氣自我療癒計時器

一個簡約日系風格的Reiki計時器應用程式，用於引導完成臼井靈氣（Usui Reiki）的自我療癒練習。

## 功能特色

- **雙模式選擇**：
  - 完整自我療癒（十二手式）
  - 脈輪啟動（七脈輪）
- **彈性時間設定**：可調整每個手式的停留時間（分鐘+秒數，以10秒為單位），預設為1分鐘
- **頌缽提示音**：每完成一個手式會響起和諧的頌缽聲
  - 可調整音量
  - 可開關
  - 支援試聽功能
- **背景音樂**：一鍵打開 YouTube Music 冥想音樂播放清單
- **進度追蹤**：視覺化顯示整體進度和當前手式進度
- **簡約設計**：日系極簡風格，幫助你專注於療癒過程

## 使用方法

1. 在瀏覽器中打開 `index.html` 文件
2. 選擇療癒模式（完整自我療癒 或 脈輪啟動）
3. 選擇每個手式想要停留的時間（分鐘+秒數）
4. 調整音效設定：
   - 可開關頌缽提示音，點擊「▶」試聽
   - 可點擊 YouTube 按鈕在新視窗播放背景音樂
5. 點擊「開始療癒」按鈕
6. 跟隨指引，將雙手放在對應的位置
7. 聆聽頌缽聲，進入下一個手式

## 療癒模式

### 完整自我療癒（十二手式）

1. 前額（第三眼）
2. 左右雙耳
3. 後腦（後腦勺）
4. 喉嚨
5. 心臟
6. 太陽輪（胃部）
7. 臍輪（丹田/腹部）
8. 底輪（前面）
9. 左右肩膀
10. 胃輪（背面）
11. 臍輪（背面）
12. 底輪（背面）

### 脈輪啟動（七脈輪）

1. 頂輪（頭頂/百會）
2. 眉心輪（第三眼）
3. 喉輪（喉嚨）
4. 心輪（心臟）
5. 太陽輪（胃部）
6. 臍輪（丹田）
7. 海底輪（底輪）

## 技術實現

- 純HTML、CSS、JavaScript實現
- 使用HTML5 Audio播放真實頌砵音效
- 響應式設計，支持各種屏幕尺寸（桌面版大圓圈，手機版小圓圈）
- 完全離線運行（背景音樂除外）
- 移動端優化：支援震動反饋和視覺提示

## 注意事項

- 建議在安靜的環境中使用
- 首次使用時瀏覽器可能需要權限才能播放音效
- 背景音樂會在新視窗/App 中播放，可與計時器同時使用
