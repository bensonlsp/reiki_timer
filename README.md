# 臼井靈氣十二手式自我療癒計時器

一個簡約日系風格的Reiki計時器應用程式，用於引導完成臼井靈氣（Usui Reiki）的十二手式自我療癒練習。

## 功能特色

- **十二手式引導**：完整的臼井靈氣十二手式流程
- **彈性時間設定**：可調整每個手式的停留時間（分鐘+秒數，以10秒為單位）
- **治癒鐘聲**：每完成一個手式會響起和諧的鐘聲
- **進度追蹤**：視覺化顯示整體進度和當前手式進度
- **背景音樂引導**：提供YouTube音樂連結，在新視窗播放背景音樂
- **簡約設計**：日系極簡風格，幫助你專注於療癒過程

## 使用方法

1. 在瀏覽器中打開 `index.html` 文件
2. （可選）點擊「在新視窗打開 YouTube 音樂」播放背景音樂
3. 選擇每個手式想要停留的時間（分鐘+秒數）
4. 點擊「開始療癒」按鈕
5. 跟隨指引，將雙手放在對應的位置
6. 聆聽鐘聲，進入下一個手式

## 十二手式說明

1. 前額（第三眼）
2. 左右雙耳
3. 後腦（後腦勺）
4. 喉嚨
5. 心臟
6. 太陽輪（胃部）
7. 臍輪（丹田/腹部）
8. 底輪（前面）
9. 左右肩膀
10. 胃輪（背面）
11. 臍輪（背面）
12. 底輪（背面）

## 技術實現

- 純HTML、CSS、JavaScript實現
- 使用HTML5 Audio播放真實頌砵音效
- 響應式設計，支持各種屏幕尺寸（桌面版大圓圈，手機版小圓圈）
- 完全離線運行（音樂除外）
- 移動端優化：支援震動反饋和視覺提示

## 背景音樂說明

- 音樂在新視窗的YouTube播放
- 你可以在音樂播放時切換回計時器頁面
- 兩個視窗可以同時運行，互不干擾

## 注意事項

- 建議在安靜的環境中使用
- 首次使用時瀏覽器可能需要權限才能播放鐘聲音效
- 計時器會持續運行，不受視窗切換影響
